# Product Images

Drop product photos here using the product ID as the filename.

## How it works

When a user views your site, the JavaScript automatically tries to load:
`/images/products/{product-id}.jpg`

If the file exists, it shows that real photo on the product card AND in the quick-view drawer.
If the file doesn't exist, the site falls back to the letter placeholder. No code changes needed.

## Recommended specs

- **Size:** 800×880 pixels (1:1.1 aspect ratio matches the card)
- **Format:** JPG (smaller file size) or PNG (if transparency needed)
- **Background:** Plain off-white/cream/dark to match brand palette
- **File size:** Keep under 200 KB per image for fast loading

## Quick photography tips

- Natural light works best — near a window, no direct sun
- Lay flat against a clean background, or hang on a hanger
- Take from straight-on, no extreme angles
- Edit on phone with a free app like Snapseed to crop and brighten

You don't need to add ALL the images at once. Just upload the ones you have. The site handles missing images gracefully.
